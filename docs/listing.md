## Name
ChatGPT Exporter

## Short Description
Easily export your entire ChatGPT conversation history into portable, archival-quality formats — Markdown, HTML, JSON, PNG, and plain text.

## Long Description
**ChatGPT Exporter** is a powerful userscript that allows you to easily export your ChatGPT conversations into various portable and archival-quality formats. 

OpenAI's web interface doesn't natively provide easy ways to export to Markdown, HTML, or capture screenshots, making it hard to save, search, and share your chat logs. ChatGPT Exporter bridges that gap. It runs entirely in your browser, authenticates via your existing session, and produces local files. Your data never leaves your machine!

**Features:**
- 📝 **Markdown Export**: Export to `.md` with YAML front matter, LaTeX preservation, and support for standard Markdown syntax.
- 🌐 **HTML Export**: Save as a self-contained `.html` file with dark/light mode support and KaTeX LaTeX rendering.
- 💾 **JSON Export**: Export raw API format matching OpenAI's official schema, plus specialized formats like TavernAI and Ooba.
- 📸 **PNG Screenshot**: Capture the entire conversation thread as a crisp image using html2canvas.
- 📋 **Plain Text**: Strip out formatting and copy plain text directly to your clipboard.
- 📦 **Bulk Export**: Download all or selected conversations in one ZIP archive.
- 🌍 **Multilingual UI**: Fully localized in English, Spanish, French, Indonesian, Japanese, Russian, Turkish, Simplified Chinese, and Traditional Chinese.
- 🕰 **Timestamps**: Display and export per-message timestamps.

## Install / Upgrade Copy
**Install / Upgrade ChatGPT Exporter**
Add ChatGPT Exporter to your browser today and take full control of your conversation data!
- ⚡ **No Accounts, No Cloud**: Runs completely locally in your browser.
- 🚀 **Updates Frequently**: We regularly update to maintain compatibility with ChatGPT's UI changes and add new features.
