⬇ Install userscript Get Tampermonkey first
Prerequisite: a userscript manager such as Tampermonkey or Violentmonkey. With one installed, clicking Install userscript opens the install prompt automatically.
  1. Install a userscript manager — Tampermonkey (Chrome/Edge/Firefox/Safari) or Violentmonkey.
  2. Click “Install userscript” above — your manager intercepts the .user.js file and shows an install dialog.
  3. Open ChatGPT at chatgpt.com — the export toolbar appears in the sidebar.
